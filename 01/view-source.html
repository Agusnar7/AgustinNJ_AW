Cómo ver la fuente de esta página

Si va al menú Ver de su navegador web, verá una opción que le permite ver el código fuente detrás de una página web.

Cuando la web despegó por primera vez, esta era una forma muy popular para que las personas aprendieran cómo se habían creado las páginas web.

Hoy en día, las personas siguen viendo la fuente de las páginas para aprender cómo se ha creado una página web.
